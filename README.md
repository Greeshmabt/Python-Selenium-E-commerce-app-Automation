# Python-Selenium-E-commerce-app-Automation

#Overview
Automated test suite for E-commerce application using Python, Selenium WebDriver, and Pytest.

#Features
User login to the application
Selects gadgets from Shopping cart through Data Driven approach
Adds items to Cart
Shows Order summary
Selects Shipping Address to various countries 
Completes checkout
Uses Pytest for framework development
HTML Report generation
Automatically captures Screenshot under test failure

#Prerequisites
Java 25
Python 
Chrome
Setup
Clone or unzip the project

#Run:
Report will be at reports/reports.html

#Structure
Page Object Model by using fixtures and parametrize concept
Data driver approach for dynamic user inputs
hooks/- For Extend report concept
